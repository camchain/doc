# Cam 节点介绍                               

全节点（full nodes）是存储 Cam 区块链全部数据的节点，通过 P2P 的方式与区块链网络连接，在区块链网络中，所有的全节点都是平等的，既充当客户端又充当服务器。

Cam 有两个全节点程序：

- Cam-wallet: 面向普通用户，提供图形界面，具有除共识外的所有功能。


- Cam-server: 面向开发者，提供命令行界面，具有一些钱包操作的基本功能，除此之外还会对外提供 API，可以与其它节点达成共识，参与区块的生成。
